The current view metadata is loaded into record-index state in `packages/twenty-front/src/modules/object-record/record-index/hooks/useLoadRecordIndexStates.ts`, then turned into query params by `record-index/hooks/useFindManyRecordIndexTableParams.ts`. The table wrapper path is `record-index/components/RecordIndexTableContainer.tsx` -> `record-table/components/RecordTableWithWrappers.tsx`, and per-cell field context rendering is in `record-table/record-table-cell/components/RecordTableCellFieldContextGeneric.tsx` with `RecordTableCell.tsx` rendering the field display.
