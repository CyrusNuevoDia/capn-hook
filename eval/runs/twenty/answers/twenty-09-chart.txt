Manual command-menu triggering starts in `buildTriggerWorkflowVersionPayloads.ts` and `TriggerWorkflowVersionEngineCommand.tsx`, which call `useRunWorkflowVersion.tsx` / `runWorkflowVersion.ts` to hit the GraphQL mutation in `workflow-trigger.resolver.ts`. The REST webhook path is `workflow-trigger.controller.ts`; both paths funnel through `workflow-trigger.workspace-service.ts` into `workflow-runner.workspace-service.ts`, which creates an `ENQUEUED` workflow run and queues `RunWorkflowJob` handled in `run-workflow.job.ts`.
