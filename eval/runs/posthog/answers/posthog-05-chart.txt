The frontend path is `frontend/src/queries/nodes/InsightViz/InsightViz.tsx` handing `query.source` plus overrides into `frontend/src/queries/nodes/DataNode/dataNodeLogic.ts`, then `frontend/src/queries/query.ts` and `frontend/src/lib/api.ts` POSTing to `/query` with `filters_override` and `variables_override`.

The backend dispatch is in `posthog/api/query.py` and `posthog/api/services/query.py`, which process the request and run `process_query_model`; `posthog/hogql_queries/query_runner.py` unwraps `InsightVizNode` to its `source` and applies dashboard filter / variable overrides before runner execution.
