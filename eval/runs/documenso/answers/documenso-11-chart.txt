Two-factor setup for account security starts in `apps/remix/app/routes/_authenticated+/settings+/security._index.tsx` via `apps/remix/app/components/forms/2fa/enable-authenticator-app-dialog.tsx`, which calls the auth client setup/enable actions. The server implementation is in `packages/auth/server/routes/two-factor.ts`, backed by `packages/lib/server-only/2fa/setup-2fa.ts`, `enable-2fa.ts`, `validate-2fa.ts`, and `verify-2fa-token.ts`.
