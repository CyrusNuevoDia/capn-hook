Paid subscription checkout is started from the API layer in `packages/trpc/server/enterprise-router/create-subscription.ts`, where `createSubscriptionRoute` calls `createCheckoutSession(...)`. A second API path starts checkout during paid organisation creation in `packages/trpc/server/organisation-router/create-organisation.ts`; both delegate to `packages/ee/server-only/stripe/create-checkout-session.ts`, which calls `stripe.checkout.sessions.create(...)`.
